# 🔍 SQL Murder Mystery — Beginner's Walkthrough

### A 26-slide presentation that teaches core SQL concepts by solving a murder.

This project is a guided walkthrough of the SQL Murder Mystery by Knight Lab, an interactive puzzle where you query a real database to identify a killer. 
The presentation is designed so that someone with zero SQL experience can follow along, learn the fundamentals, and solve the mystery from start to finish.

### What's covered

- `SELECT` and `WHERE` — retrieving and filtering data
- `JOIN` — connecting tables to follow the evidence
- `GROUP BY` / `COUNT` — aggregating clues to find patterns

### About this project

This presentation was created as a deliverable for the SQL module of the **UC Clermont Data Club for HIM Students**, led by **Jesus Calvo** — creator and lead instructor of the program. The Data Club was completed as part of my senior practicum experience.

The goal was to teach by doing: rather than a dry syntax reference, the deck walks through each concept in the context of actually solving the mystery. Every query has a reason, and the stakes are (fictionally) high.
